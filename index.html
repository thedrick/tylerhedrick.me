Projects

Chefmates

Chefmates was born as a web application during the Facebook Hackathon at CMU, but has now become a personal project to create an iPhone equivalent of that app. The idea is to put in the ingredients you have in your pantry, and get suggestions of recipes you can make with those ingredients. You will be able to share those recipes with your Facebook friends, as well as add your own recipes to your cookbook.

Archer's Revenge

Archer's Revenge is a 2-player turn-based strategy game I made in a small group for a class project. The game is written entirely in javascript using HTML5 canvas. This was my first time using javascript, and received first place in a class-wide competition for the best game. You can play the game here.

Lien

Lien (meaning "connection" in French) is a native iOS application built during the Tartanhacks Hackathon 2013. With Lien it is very simple to set up impromptu events and invite your friends, or start a public party for people to join. I like to think of it as Twitter meets Facebook Events.

Gilbert

Gilbert evolved out of frustration found while building Lien with how tedious it can be to build custom views for mobile applications. With Gilbert, the user can write a short JSON object describing a view in a mobile app on the left, and have it be converted into both Objective-C and HTML/CSS code to render that view on the right. Gilbert was created during MHacks 2013 at University of Michigan.
